Implement in Your Roblox Game

Copy this code snippet to your Roblox script:

-- Key authentication system local HttpService = game:GetService("HttpService") -- Update this URL to your GitHub Pages site local keysUrl = "https://website-tree.github.io/verfrblx/keys.json" function validateKey(key) local success, result = pcall(function() -- Fetch the keys database local response = HttpService:GetAsync(keysUrl) local keyData = HttpService:JSONDecode(response) -- Check if key exists in the valid keys list if keyData and keyData.validKeys then for _, validKey in ipairs(keyData.validKeys) do if key == validKey then return true end end end return false end) return success and result or false end

Implementation Instructions

Follow these steps to set up key authentication in your Roblox game

1 Generate and Save Keys

Start by generating unique keys in the Generator tab. Each key can be used to authenticate a user in your game.

  1. Go to the Generator tab
  2. Click "Generate New Key" to create a unique key
  3. Click "Save Key" to store it for validation
  4. Test the key using the "Test Key" button

2 Download and Host keys.json

Your generated keys are stored in a JSON file that you'll need to host on GitHub Pages.

  1. Go to the Keys tab
  2. Click "Download keys.json" to save your keys file
  3. Create a GitHub repository (e.g., verfrblx) if you don't have one already
  4. Upload the keys.json file to your repository
  5. Enable GitHub Pages in your repository settings

3 Implement in Roblox

Copy the provided Lua code to your Roblox game to validate keys against your hosted keys.json file.

  1. Copy the code snippet from the Generator tab
  2. Create a script in your Roblox game (e.g., in ServerScriptService)
  3. Paste the code and update the URL to your GitHub Pages URL
  4. Add your own logic for what happens when a valid key is entered
  5. Enable HTTP Requests in your game settings (Game Settings → Security → Allow HTTP Requests)
