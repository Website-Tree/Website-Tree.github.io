Implementation Instructions

How to use the key authentication in your Roblox scripts

1 Generate a Key

First, generate a unique key using the Key Generator tab. This key will be used to authorize your Roblox script.

2 Implement the Script

Copy the following Lua script into your Roblox project and replace YOUR_KEY_HERE with your generated key:

-- Key Authentication Module local KeyAuth = {} -- Replace with your generated key local key = "YOUR_KEY_HERE" -- Validation URL (keep this URL) local validationUrl = "https://your-replit-url.repl.co/api/keys/validate" function KeyAuth.ValidateKey() local success, isValid = pcall(function() local HttpService = game:GetService("HttpService") -- Prepare the request data local requestData = { key = key } -- Convert data to JSON local jsonData = HttpService:JSONEncode(requestData) -- Send the request to validate the key local response = HttpService:PostAsync( validationUrl, jsonData, Enum.HttpContentType.ApplicationJson ) -- Parse the response local responseData = HttpService:JSONDecode(response) return responseData.valid end) if success and isValid then -- Key validated successfully return true else -- Key validation failed return false end end -- Example usage function KeyAuth.Start() local isKeyValid = KeyAuth.ValidateKey() if isKeyValid then -- Key is valid, enable your script functionality print("Key is valid! Unlocking features...") return true else -- Key is invalid, disable functionality print("Invalid key! Access denied.") return false end end return KeyAuth

3 Using the Module

Here's how to use the key authentication module in your Roblox script:

-- In your main script local KeyAuth = require(script.Parent.KeyAuth) -- Check if the key is valid before running your code if KeyAuth.Start() then -- Your protected code here print("Script is running with valid authentication!") -- Example premium feature local function createPremiumItem() -- Your premium item creation code end createPremiumItem() else -- Handle invalid key print("Authentication failed! Premium features are disabled.") end

4 HTTP Requests in Roblox

Remember to enable HTTP requests in your Roblox game:

  1. Go to Game Settings
  2. Select the Security tab
  3. Enable "Allow HTTP Requests"
  4. Add your validation URL to the allowed list

5 Deploy Your API

Make sure to update the validationUrl in the script with your actual deployed URL.
